# 갱년기 자가진단표 — 마갱탐

**마루이모의 갱년기 탐구생활(마갱탐)**에서 제작한 갱년기 자가진단표입니다.

👉 **[진단표 바로 사용하기](https://maru-imo.github.io/menopause-check/)**

---

## 이 진단표는 무엇인가요?

갱년기를 직접 겪고 있는 마루이모가 공부하며 만든 자가점검 도구입니다.
지난 한 달 동안 경험한 증상을 체크하면 점수와 함께 간단한 해석을 제공합니다.

20가지 증상 항목을 0~3점으로 평가하며, 총점 60점 만점입니다.
점수 해석 기준은 호주폐경학회(AMS) 자료를 참고했습니다.

---

## 참고 및 주의사항

- 이 진단표는 갱년기 증상을 **스스로 파악하기 위한 참고용**입니다.
- 의학적 진단 도구가 아닙니다. 점수가 높다고 해서 반드시 갱년기인 것은 아닙니다.
- 15점 이상이라면 병원 상담 시 이 표를 출력해서 가져가는 것을 권합니다.
- 진단과 치료는 반드시 전문의와 상담하세요.
- 매달 체크해서 변화를 기록해두면 진료 시 유용하게 활용할 수 있습니다.
